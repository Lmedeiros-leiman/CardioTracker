# cardio-tracker-patient-registry

Patient registry package for the CardioTracker monorepo.

## Structure

```text
src/
  domain/
    entities/
    value-objects/
    enums/
    services/
    repositories/
  application/
    use-cases/
    dto/
    ports/
  infra/
    repositories/
    mappers/
```

Use `bun install` from the repository root to install workspace dependencies.
